abstract fun <T : ModalResult> openModal(modal: Modal<T>)
Content copied to clipboard

Opens a new Modal window on top of the Screen. A modal window blocks access to all Components on this Screen and also stops component events until the modal window is closed. A Modal returns a ModalResult which represents the result of opening the Modal itself.
